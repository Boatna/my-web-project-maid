# ระบบบันทึกการตรวจงานแม่บ้าน

ระบบสำหรับหัวหน้างานใช้บันทึกการตรวจงานแม่บ้าน พร้อมถ่ายรูปจุดที่ไม่เรียบร้อย

## คุณสมบัติ

- ✅ เลือกรายชื่อหัวหน้า
- ✅ เลือกแม่บ้านที่ตรวจ
- ✅ ระบุพื้นที่และจุดตรวจ
- ✅ ให้คะแนนผลการตรวจ (เรียบร้อย/ปรับปรุง/ไม่เรียบร้อย)
- ✅ ถ่ายรูปจากกล้อง (ไม่อนุญาตอัพโหลดไฟล์)
- ✅ บันทึกประวัติการตรวจ
- ✅ สร้างรายงานสรุป
- ✅ ทำงานออฟไลน์ได้
- ✅ PWA (ติดตั้งบนมือถือได้)

## การใช้งานบน GitHub Pages

1. **Fork repository นี้** ไปยัง GitHub ของคุณ
2. **ตั้งค่า GitHub Pages**:
   - ไปที่ Settings > Pages
   - ตั้งค่า Source เป็น GitHub Actions หรือ main branch
3. **เข้าถึงเว็บแอป** ที่ `https://[username].github.io/[repository-name]`

## การใช้งาน

### สำหรับหัวหน้างาน:
1. เลือกชื่อหัวหน้า
2. ไปที่หน้า "ตรวจงาน"
3. เลือกแม่บ้านและพื้นที่ที่ตรวจ
4. ถ่ายรูปจุดที่ไม่เรียบร้อย
5. ให้คะแนนผลการตรวจ
6. บันทึกข้อมูล

### ดูประวัติ:
- ดูประวัติการตรวจทั้งหมด
- กรองตามวันที่หรือแม่บ้าน
- ดูรูปภาพที่บันทึก

### สร้างรายงาน:
- สรุปรายงานตามช่วงเวลา
- ดูประสิทธิภาพแม่บ้าน
- วิเคราะห์จุดที่มีปัญหา
